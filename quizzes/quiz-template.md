## Quiz Template

This is a template for creating quizzes for each module of the Physical AI & Humanoid Robotics textbook.

### Instructions
- Answer all questions to complete the quiz
- Each quiz is designed to validate your understanding of the module content
- Aim for a score of 80% or higher to ensure adequate comprehension

### Question Format
1. Multiple choice questions with 4 options
2. True/False questions
3. Short answer questions (where appropriate)

### Example Question:
**What does ROS 2 stand for?**
- A) Robot Operating System 2
- B) Robotic Operations Suite 2
- C) Robot Operating Software 2
- D) Robotic Operating System 2

Answer: A
