# Citation Guide

## APA Style for Physical AI & Humanoid Robotics Textbook

This document outlines the APA citation style to be used throughout the textbook as required by the project constitution.

### General Guidelines

- All sources must be cited using APA 7th edition format
- In-text citations follow the (Author, Year) format
- References should be listed alphabetically by author's last name
- All borrowed content must be properly attributed

### Common Citation Formats

#### Books
Author, A. A. (Year). *Title of work*. Publisher.

#### Journal Articles
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. *Title of Periodical*, volume(issue), pages. https://doi.org/xx.xxx/yyyy

#### Web Resources
Author, A. A. (Year, Month Date). Title of webpage. *Website Name*. URL

#### Technical Documentation
Organization Name. (Year). *Title of documentation*. URL

### Examples for Robotics/AI Content

- When citing ROS 2 documentation: ROS Development Team. (2023). *ROS 2 Documentation*. https://docs.ros.org/
- When citing NVIDIA Isaac: NVIDIA Corporation. (2023). *NVIDIA Isaac Documentation*. https://nvidia-isaac.readthedocs.io/
- When citing academic papers: Include DOI when available

### Implementation Notes

- All chapters should include proper citations for any referenced material
- Code examples should attribute original sources when adapted
- Diagrams and figures should be properly attributed
- External libraries and frameworks should be cited appropriately
